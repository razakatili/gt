Membuat Desain Website Sekolah Menggunakan Bahasa HTML Dan CSS

Nazriel Ilham 20:20:00
Nazriel Ilham
Kini cukup banyak pihak sekolah membangun website untuk membuka jalur komunikasi kepada para murid dengan memberikan informasi update mengenai sekolahnya. Selain itu, website sekolah dapat membantu memberikan informasi nilai siswa dan dapat dipantau langsung oleh orang tua/wali melalui situs website tersebut. Pembuatan website sekolah diharapkan mampu memberikan update informasi, seperti: kegiatan ekstrakulikuler, update organisasi siswa, jadwal sekolah, nilai siswa dan lain sebagainya. Namun untuk membuat website sekolah, kebanyakan pihak sekolah membayar orang yang ahli atau yang mengerti di bidangnya untuk membuat website sekolah. Namun bagi yang ingin membuat website sekolah, Kami ingin berbagi informasi mengenai pembuatan desain website sekolah. Berikut ini cara pembuatan website sekolah beserta dengan script HTML dan CSS. 

1. Script HTML

2. Script CSS 
Buat form baru menggunakan Notepad++ dan simpan dengan ekstensi .css. Anda dapat membuat style pada elemen HTML menggunakan script CSS, seperti: desain huruf, warna background, jenis tulisan, tampilan website dan lain sebagainya.

3. Hubungkan HTML ke CSS
